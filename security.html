Security & Data Handling

Data Hosting

All data is processed within UK-based infrastructure.

Data Collection

Data is processed only when explicitly initiated by the user. No background data collection occurs.

Storage

Data is processed on a session basis and is not retained long-term.

Security Measures

  • Encryption in transit (HTTPS/TLS)
  • Secure processing environment
  • Access controls applied
  • Minimal operational logging

AI Usage

AI is used for structured extraction and summarisation only.

No user data is used for model training or shared with third parties.

This system supports human decision-making and does not make autonomous decisions.
