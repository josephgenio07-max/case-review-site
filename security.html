Security & Data Handling

Data Hosting

All data is processed within UK-based infrastructure.

Data Collection

Data is only processed when explicitly initiated by the user.

Storage

No long-term storage. Data is processed per session.

Security Measures

  • HTTPS encryption in transit
  • Encrypted processing environment
  • Access controls
  • Minimal logging

AI Usage

No user data is used for model training or shared with third parties.

This system supports human decision-making and does not make autonomous decisions.
